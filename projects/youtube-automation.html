Under the Hood

Pipeline Deep-Dive

1

discover.py — Trend Discovery

Scans trending channels and viral content feeds. Filters clips by engagement ratio (likes, comments, shares vs. views) to identify high-potential content. Only selects clips under 60 seconds that are optimized for the Shorts format. Outputs a ranked queue of candidate videos for download.

2

yt-dlp — Content Download

Uses yt-dlp to download the selected clips at the highest available quality. Handles rate limiting, retries, and format selection automatically. Downloads are stored in a staging directory with metadata preserved for the processing pipeline.

3

process.py — FFmpeg Processing

The core processing engine powered by FFmpeg. Applies CRF 20 quality encoding at 4Mbps target bitrate using h264 codec. Crops to 1080×1920 vertical format, applies a 1.05x speed boost for pacing, AI color grading for visual pop, and encodes AAC audio at 128kbps. Each video is polished and platform-ready in seconds.

4

upload.py — YouTube Publishing

Authenticates with YouTube Data API v3 and uploads each processed video. Generates AI-powered titles and descriptions optimized for search. Adds growth hashtags, selects the best category, and sets thumbnail preferences. Each upload is tagged and tracked for performance analytics.
